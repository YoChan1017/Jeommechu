> Java · Servlet · JDBC · MySQL · HTML · CSS · WAR Deployment

---

## 프로젝트 개요

**Jeommechu**(점메추)는 사용자 선택 조건을 기반으로 점심 메뉴를 추천하는 Java Servlet 기반 웹 애플리케이션입니다.
 
**Java 웹 애플리케이션 구조 이해**, **JDBC 기반 데이터베이스 연동**, **서블릿 중심의 서버 사이드 로직 구현**을 목표로 개발되었습니다.

사용자는 웹 화면을 통해 로그인 후 메뉴 목록을 확인하고,  
저장된 점심 메뉴 데이터를 기반으로 메뉴 추천 또는 선택 기능을 이용할 수 있습니다.

---

## 개발 목적

- Java Servlet 기반 **웹 애플리케이션 구조 이해**
- JDBC를 활용한 **MySQL 데이터베이스 연동 경험**
- DAO / VO 패턴을 적용한 **백엔드 코드 구조화**
- WAR 패키징을 통한 **웹 서버 배포 구조 학습**
